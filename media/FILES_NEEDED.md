# Необходимые файлы

Список файлов, которые нужно добавить в папки для работы игры:

## Изображения (frames/)

Скопируйте эти файлы в `media/frames/`:

- `matrix.jpg` - Кадр из фильма "Матрица"
- `inception.jpg` - Кадр из фильма "Начало"
- `pulp_fiction.jpg` - Кадр из фильма "Криминальное чтиво"
- `shawshank.jpg` - Кадр из фильма "Побег из Шоушенка"
- `godfather.jpg` - Кадр из фильма "Крестный отец"

## Видео (videos/)

Скопируйте эти файлы в `media/videos/`:

- `avengers.mp4` - Видеофрагмент из фильма "Мстители"
- `interstellar.mp4` - Видеофрагмент из фильма "Интерстеллар"
- `joker.mp4` - Видеофрагмент из фильма "Джокер"
- `parasite.mp4` - Видеофрагмент из фильма "Паразиты"
- `dune.mp4` - Видеофрагмент из фильма "Дюна"

## Где найти файлы

Вы можете:
1. Скачать кадры из фильмов (например, с сайтов вроде IMDb, или сделать скриншоты)
2. Скачать короткие видеофрагменты (trailers или clips)
3. Использовать свои собственные медиафайлы (убедитесь, что у вас есть права)

## После добавления файлов

Запустите скрипт загрузки:

```bash
# Windows
.\scripts\upload-media.ps1

# Linux/Mac
./scripts/upload-media-docker.sh
```

Или загрузите вручную через MinIO Console: http://localhost:9090
